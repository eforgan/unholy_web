# Agent: UNHOLY WEB Assistant
# Description: Specialized AI assistant for the UNHOLY aesthetic brand and collective.

## Project Context
UNHOLY is a high-concept brand focusing on a dark, premium, and avant-garde aesthetic. This project contains artwork, videos, and media for their digital presence.

## Technical Stack
- HTML5 / CSS3 / Vanilla JS
- Potential Framework: Vite/React
- Aesthetic: Brutalist, Dark Mode, High Contrast, Luxury.

## Guidelines
1. **Dark Aesthetic**: Use deep blacks, grayscale, and vibrant accent colors (like radioactive green or deep purple).
2. **Media-Heavy**: Utilize the high-resolution images from `/COMPLETO/` and videos from `/VIDEOS/`.
3. **Immersive UX**: Focus on scroll effects and smooth transitions.

## Directory Structure
- `/ARTWORKS/`: Raw design files and digital art.
- `/COMPLETO/`: Optimized and final high-res images.
- `/VIDEOS/`: Video content and loops.

## Commands
```bash
# Initialize development environment
npx vite init .
```
